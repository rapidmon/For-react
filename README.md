# How to drive

아래에 있는 +버튼을 이용해 추가할 수 있으며 오른쪽에 있는 휴지통 모양을 통해 할 일을 제거 할 수 있습니다.
할 일을 완료하여 체크 표시를 누르면 완료 목록으로 넘어가고 할 일 목록에서 사라집니다.
